# Roast Sounds

This directory contains short audio clips that play when the app catches you slacking off.

## Included sounds

| File | What it says |
|------|-------------|
| caught_slacking.aiff | "Are you serious right now? Get back to work!" |
| phone_detected.aiff | "Put the phone down! I can see you!" |
| disappointed.aiff | "I'm not mad, I'm just disappointed." |
| bruh.aiff | "Bruh. What are you even doing right now?" |
| focus_up.aiff | "Focus up! You said you'd be productive today!" |
| wow.aiff | "Wow. Just wow. This is what we're doing instead of working?" |

## Add your own

Drop any `.mp3`, `.wav`, `.aiff`, `.ogg`, or `.m4a` file into this folder and the app will automatically pick it up. The app selects a random clip each time, so the more you add, the more variety you get.
